Presentación

Bienvenido al Objeto Virtual de Información O.V.I. Donde Identificara los diferentes modelos y requisitos del proceso en el desarrollo de Software, conociendo las ventajas y desventajas de cada modelo para el desarrollo de software.
También aprenderá a escoger el modelo más idóneo para el desarrollo de un software específico, logrando distinguir cada paso dentro del proceso de desarrollo de software.
TPodrá visualizar dos tipos de contenido que son material de lectura y material multimedia (videos).
